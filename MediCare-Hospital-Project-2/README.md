# 🏥 MediCare Hospital Project

## 📌 Overview
MediCare Hospital is a responsive hospital website built using HTML, CSS, JavaScript, and Node.js (Express backend).

It is developed as part of the Decode Labs Internship Program to demonstrate frontend + backend development skills.

---

## 🚀 Live Demo
https://haniya-ramzan.github.io/DecodeLabs-Internship/MediCare-Hospital-Project/

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- JSON (File-based storage)

---

## 📂 Project Structure

```
MediCare-Hospital-Project/
│
├── backend/
│   ├── controllers/
│   ├── routes/
│   ├── data/
│   └── server.js
│
├── images/
├── index.html
├── style.css
├── script.js
└── README.md
```

---

## ✨ Features

- Responsive hospital website
- Doctor listing system
- Appointment form
- Contact form
- Backend REST API (GET, POST)
- JSON-based data storage
- Form validation on server side

---

## ⚙️ How to Run Locally

### Frontend
Just open:
```
index.html
```

---

### Backend

```bash
cd backend
npm install
node server.js
```

Server runs at:
```
http://localhost:3000
```
