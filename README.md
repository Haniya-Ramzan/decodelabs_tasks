# MediCare Hospital Management Website

## 📌 Project Overview
MediCare Hospital Management Website is a responsive hospital website developed using HTML, CSS, JavaScript, Node.js, and Express.js. It provides users with hospital information, doctor profiles, appointment booking, contact services, blogs, and a simple backend API.

## ✨ Features

- Responsive Hospital Website
- Appointment Booking Form
- Contact Form
- Doctor Profiles
- Hospital Departments
- Patient Testimonials
- Blog Section
- Gallery
- Backend using Node.js & Express.js
- JSON-based data storage

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

## 📂 Project Structure

MediCare-Hospital-Project-2/
│── backend/
│── images/
│── index.html
│── style.css
│── script.js
│── package.json
│── README.md

## ⚙️ Backend Setup

```bash
cd backend
npm install
npm start
```

Server runs on:

```
http://localhost:3000
```
